# Flappy Bird
Reproduction of one of the most popular arcade-style game of 2014.

# Requirements

### pygame

# Installation
1. clone [FlappyBird repo](https://github.com/pumassv1/FlappyBird)
2. cd into this folder.
3. run main.py in your IDE
4. Have fun!
